# VV UI
`SF-UI` 是一款基于 `Vue.js 2.0` 的前端 UI 组件库，主要用于快速开发web中后台产品

## 特性

- 基于 `Vue2.0` 开发的 UI 组件

## 安装

- npm 

```bash
npm install vvui --save
```

## 浏览器支持

- 现代浏览器和 IE9 及以上

## 使用

```js
import SfUI from 'sf-ui' // 引入组件库
import 'sf-ui/packages/theme-default/lib/index.css' // 引入样式库

Vue.use(SfUI)
```
